Known Issues:
	Collecting a locust chest containing a Max Locust Up gives you the locusts first and then upgrades your capacity, wasting locusts if you're at max capacity.
	It is theoretically possible that logic will require you to loop Cif's Domain (or some other subset of floors) more times than reasonable in order to grind up locusts.



v0.10.1
	Change:
		Reworked Connection Menu navigation & other misc changes
			Now pressing "Enter" is required to type in a field
			To connect press "Enter" on the submit button
			"Tab" still navigates between fields
	Fix:
		Checked locations are now queued
			Aka it's now safe to check locations while not connected to AP
			The game will track what needs to be sent and will send them out when you reconnect to AP
		End Run now disconnects you from AP
		Receiving a shortcut from AP in the room with that shortcut no longer requires reloading to room to use it
		Fixed status line on Connection Menu not showing the proper connection status



v0.10.0

	New:
		Smiler and Interface pathfinding logic is now functional
		Added Palette Menu and custom Palette Functionality
			Game reads new palettes.txt file in save directory
			Specify a name, and four hex colors (0x123456), up to nine custom palettes at once
		Added "Extra AP Menu" to load less important menus
			Contains Connection Menu, Palette Menu, and Debug Menu
			Connection menu is opened by default until connected to AP
	
	Change:
		Locusts are no longer removed on death
			As a result of this, B077's Chest Score is now 3 instead of 2
		Remove ProgressiveBrands and ShortcutCheating options
			Logic rewrites, waypoints, and free deaths render these options obsolete
		LocustCapacityUp has been disabled and forced to 3
			There were generation issues with lower values
			This setting is likely to be removed in the future

	Fix:
		Max locust score is now calculated correctly
		Shortcut 4 now checks for Killers
		Taking the Locust Tile from the Interface will no longer revert your locusts
		Using a waypoint immediately after taking the void rod no longer breaks the music
